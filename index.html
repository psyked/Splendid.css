Splendid.css

Delightful defaults for humdrum HTML, using a little CSS sparkle.

The default appearance of HTML elements is a little dull, we shouldn't have to worry about adding custom css classes or changing the markup of elements in our applications to make them fit the expectations of a complex framework.

Splendid.css offers a new set of styles for the basic HTML elements, with delightful details baked in to the core, important basics like accessibility, and all with minimal modification to your existing HTML, so there's no need to go back to retrofit your HTML with a whole slew of new CSS classes.
